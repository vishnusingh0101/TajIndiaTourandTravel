# Taj India Tour and Travel

A modern, responsive website for Taj India Tour and Travel - showcasing premium tour packages across India's most iconic destinations including Agra, Mathura, Vrindavan, and Jaipur.

## Features

- **Responsive Design**: Mobile-first approach with Bootstrap framework
- **Tour Packages**: Customizable packages for popular Indian destinations
- **SEO Optimized**: Full meta tags, structured data, and sitemap
- **Contact Forms**: Easy booking and inquiry submission
- **WhatsApp Integration**: Direct messaging capability
- **Performance Optimized**: Fast loading times with optimized assets

## Technology Stack

- **Frontend**: HTML5, CSS3, Bootstrap 4/5, JavaScript
- **Hosting**: Firebase Hosting
- **Server**: Express.js
- **Analytics**: Google Tag Manager & Google Analytics

## Installation & Development

### Prerequisites
- Node.js (v14 or higher)
- Firebase CLI

### Local Development

```bash
# Install dependencies
npm install

# Serve locally
firebase serve

# Open http://localhost:5000 in browser
```

### Deployment

```bash
# Deploy to Firebase Hosting
firebase deploy
```
